# PKMOO-Leagues-Guide
A guide for beat 4 leagues en 1 hour

## Cómo ejecutar

Este proyecto requiere un servidor web local para funcionar correctamente debido a las políticas CORS del navegador.

### Opción 1: Usar el servidor incluido (Recomendado)

1. Ejecuta `start-server.bat` (o `node server.js` en la terminal)
2. Abre tu navegador en `http://localhost:8000`
3. La aplicación funcionará correctamente

### Opción 2: Otros servidores locales

También puedes usar cualquier servidor web local como:
- Python: `python -m http.server 8000`
- PHP: `php -S localhost:8000`
- Live Server (extensión de VS Code)
